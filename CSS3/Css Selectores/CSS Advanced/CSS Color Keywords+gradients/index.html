A palavra-chave transparente A transparentpalavra-chave é usada para tornar uma cor transparente. Isso geralmente é usado para criar uma cor de fundo transparente para um elemento.

A palavra-chave cor atual A currentcolorpalavra-chave é como uma variável que contém o valor atual da propriedade color de um elemento. Essa palavra-chave pode ser útil se você quiser que uma cor específica seja consistente em um elemento ou página.
