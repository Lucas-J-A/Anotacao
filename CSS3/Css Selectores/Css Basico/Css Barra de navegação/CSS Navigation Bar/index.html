CSS Navigation Bar

Barras de navegação Ter uma navegação fácil de usar é importante para qualquer site. Com CSS você pode transformar menus HTML chatos em barras de navegação bonitas. Barra de Navegação = Lista de Links Uma barra de navegação precisa de HTML padrão como base. construiremos a barra de navegação a partir de uma lista HTML padrão. Uma barra de navegação é basicamente uma lista de links, então usar os elementos ul e li faz todo o sentido:

  • Home
  • News
  • Contact
  • About
