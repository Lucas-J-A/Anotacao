CSS Image Sprites

Um sprite de imagem é uma coleção de imagens colocadas em uma única imagem. Uma página da Web com muitas imagens pode levar muito tempo para carregar e gerar várias solicitações de servidor. O uso de sprites de imagem reduzirá o número de solicitações do servidor e economizará largura de banda. Com CSS, podemos mostrar apenas a parte da imagem que precisamos.
